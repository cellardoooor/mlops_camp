# Module 1 Homework - Instructions

## Overview
Это домашнее задание посвящено прогнозированию длительности поездки на желтом такси NYC используя данные за январь и февраль 2023 года.

## Файлы
- `homework_solution.ipynb` - Jupyter Notebook с решением всех 6 вопросов
- `requirements.txt` - список необходимых Python пакетов

## Шаги для выполнения

### Шаг 1: Установка зависимостей
```bash
pip install -r requirements.txt
```

Или вручную:
```bash
pip install pandas numpy scikit-learn pyarrow
```

### Шаг 2: Запуск ноутбука
Откройте `homework_solution.ipynb` в Jupyter Notebook/JupyterLab или VS Code.

### Шаг 3: Проверка результатов
Ноутбук автоматически загрузит данные и выведет ответы на все 6 вопросов.

### Шаг 4: Отправка ответов
Перейдите на https://courses.datatalks.club/mlops-zoomcamp-2025/homework/hw1
